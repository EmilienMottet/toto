## Comment lancer le projet ?

Assez simple, premièrement se créer un environnement virtuel Python3 :

```bash
python3 -m venv .venv```

Ensuite, installer les dépendances :

```bash
pip install -r requirements.txt```

Puis lancer le projet :

```bash
uvicorn main:app --reload
```
